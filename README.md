# Pixi.JS Top-Down Dynamic-Loading Multiplayer Test

# Running

- Use `npm run dev` to run dev mode
- Use `npm run build` to build and `npm run preview` to serve the build

# Tools

- TypeScript
- Pixi.JS
  - Pixi.JS / Tilemap
- Krita (for assets modifications, sources available)
- Vite (dev & build)
